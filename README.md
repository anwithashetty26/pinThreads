# Pinthreads

Pinthreads is a Reddit × Pinterest inspired social discovery platform built with React.

## Features

* Pinterest-style card layout
* Reddit-style discussion threads
* Nested comments and replies
* Upvote and downvote system
* Like posts
* Save posts
* Dark theme UI
* Interactive post detail panel

## Technologies Used

* React
* JavaScript
* HTML
* CSS
* Vite

## How to Run

1. Install dependencies

npm install

2. Start the development server

npm run dev

## Project Status

This project was developed as a frontend social media platform inspired by Reddit and Pinterest.
