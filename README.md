# Pinthreads

Pinthreads is a Reddit × Pinterest inspired social discovery platform built with React.

## Features

* Post ideas
* Browse content
* Comment Section
* Modern UI
* Like posts
* Save posts
* Interactive post detail panel

## Technologies Used

* React
* JavaScript
* HTML
* CSS
* Vite

## How to Run

1. Install dependencies

npm install

2. Start the development server

npm run dev
